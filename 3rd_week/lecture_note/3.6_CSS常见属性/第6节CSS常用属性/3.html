1952年，一位名叫哈里●罗宾逊（Harry Robinson）的英国人在欧洲游历，来到了位于阿尔卑斯山下的奥地利小镇“哈尔施塔特”。他有一手绝活，那就是从他的故乡伦敦带来的蛋糕烘焙手艺。这些外酥里嫩、带着奇妙香气起司蛋糕让哈里赢得了大家的好感，居民们也毫不吝啬地在镇上的露天广场为Harry举办了一场盛大的欢迎晚会。

晚会上，哈里决定脱离传统烤法的条款限制，让想象力做主，尝试烤一批他从未做过的新蛋糕。他根据这些年在欧洲各国游历的所见所闻，每个地方特有的偏好、特殊的糕点原料，利用这些全新的元素，加入上品奶酪，他做出了花样百出、口味缤纷的新式起司蛋糕（cheese cake），在晚会上大获成功！

哈里将自己的手艺慷慨地传授给大家，善于精工细作的奥地利人则通过自己的奇思妙想将不起眼的蛋挞做的精美无比，这些各种口味、各种风格的蛋糕被游客们称为“Harry’s cheese cake”。 你不得不惊讶！一家名为“哈里老伯伯®”的蛋糕店开始在中国大陆红火起来。门头显著位置有一个拄着拐杖，拿着烟斗的白胡子老头形象，旁边哈里老伯伯”起司蛋糕工坊，欢迎大家前来品尝！
